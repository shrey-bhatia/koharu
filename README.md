# Koharu

LLM を使った自動漫画翻訳ツール。

Automated manga translation tool with LLM, written in Rust.

## Models

- [comic-text-detector](https://github.com/dmMaze/comic-text-detector) - 漫画のセリフを検出するモデル
- [manga-ocr](https://github.com/kha-white/manga-ocr) - 漫画の文字を認識するモデル

## Development

### Prerequisites

- Rust 1.85
- Bun
